## Reddit — r/EliteDangerous (self post)

**Title:** I got tired of re-Googling the same Elite stuff, so I turned my notes into a site — edblackbox.com

**Body:**

o7 commanders.

For a while now I've kept a pile of notes for myself — which ship to actually fly for a
job, how to unlock a given engineer, what a blueprint really costs in mats, where to farm
the annoying ones. Every time I came back to the game after a break I'd end up re-looking-up
the same things. So I finally cleaned the notes up, cross-linked them, and put them online.

It's called the **Black Box**. No ads, no login, no paywall — just pages.

What's in there right now:

- A dossier for basically every ship-and-role worth flying (Corvette for combat, Cutter for
  trade, AspX for exploration, etc.), with a 1–100 suitability rating and the reasoning
  behind it — not just "this one's good."
- "Best ship for the job" ladders if you'd rather start from the role than the hull.
- The full engineering side: the engineer referral tree, blueprint + module references, an
  unlock checklist, and material-farm routes.
- Game-system guides — BGS, Powerplay 2.0, colonisation, fleet carriers, combat venues — plus
  role playbooks for combat, mining, exploration, trading, passenger and AX.
- A lexicon for all the acronym soup (FSD, HRP, SLF, the works).

Two things I care about: I keep the numbers honest (every stat is checked against
coriolis-data, INARA, EDSY and EDSM, and each page lists its sources), and where something is
my opinion — a rating, a build call — I say so instead of pretending it's gospel.

It's a solo project and I'm sure I've got things wrong or missing. If you spot a bad number
or a ship that deserves better/worse, tell me — genuinely want the corrections.

Fly safe. → **edblackbox.com**
