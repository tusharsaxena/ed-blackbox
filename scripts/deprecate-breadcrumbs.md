# deprecate-breadcrumbs.py

Retires the standalone `<nav class="breadcrumbs">` strip site-wide and makes the in-header
`.hdr-crumb` the **only** breadcrumb, with its parent trail turned into navigable links.

## What it does

Every guide used to carry two breadcrumb representations:

1. a standalone strip below the header — `<nav class="breadcrumbs">` — `Home › … › Page`,
   every segment a link;
2. an in-header crumb — `.hdr-crumb` — the current page as a title over a **plain-text**
   parent trail with **no Home** and **no links**.

This script collapses them into one:

- the `.hdr-crumb` trail segments become **links**, reusing the exact `href`s + labels from
  the page's own `<nav class="breadcrumbs">` (Home is dropped — it never appears in the
  header crumb);
- the standalone `<nav class="breadcrumbs">` (plus its `<!-- ---- BREADCRUMBS ---- -->`
  comment and trailing blank line) is removed.

The breadcrumb nav is the data source: **trail** = every `<a>` in it except the first
(Home); **title** = the `.current` segment. A page with only Home + current yields an empty
trail → the `.hdr-crumb` gets the `solo` modifier and no trail span.

The matching CSS lives in `design-system/css/ed-blackbox.css`:
`.hdr-crumb-trail a{color:inherit;text-decoration:none}` + an amber `:hover`/`:focus-visible`
(links look identical to the old plain trail at rest, light on hover). The now-unused
`.breadcrumbs` component rules were deleted in the same change. The masthead top padding was
also tightened to `var(--space-13)` (symmetric with the bottom) since the nav no longer sits
in the gap above it.

## Usage

```bash
python3 scripts/deprecate-breadcrumbs.py            # all guides (excl. index.html) + DS templates
python3 scripts/deprecate-breadcrumbs.py <path...>  # specific files/dirs
python3 scripts/deprecate-breadcrumbs.py --check     # report, write nothing
```

Idempotent: a page whose `<nav class="breadcrumbs">` is already gone is skipped. The rewrite
is otherwise byte-preserving — only the `.hdr-crumb` block and the nav block change, with the
file's own indentation.

## Not touched here

- **`guides/index.html`** — generated by `scripts/generate-guides-index.sh` (its template was
  updated in the same change; re-run that script, not this one).
- **`guides/ships/ship-role-matrix.html`** — its generator `scripts/generate-ship-role-matrix.py`
  is **stale** (the live page was hand-rebuilt with a richer header the generator can't emit).
  The live page was converted by this script; do **not** re-run `generate-ship-role-matrix.py
  --page` without first re-syncing it, or the page regresses.

## Obsoleted by this change

`scripts/header-crumb-from-breadcrumbs.py` (built the `.hdr-crumb` *from* the now-removed nav)
is no longer applicable. Authoring scripts that edited the nav — `apply-role-tweaks.py`,
`add-breadcrumb-subsection.py` — operate on markup that no longer exists.

This was a one-shot migration; the script is kept in `scripts/` per repo rule 6 for the
record and in case the pattern is needed again.
