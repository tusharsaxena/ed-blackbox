# scripts/

Reusable scripts for building and maintaining the Elite:Dangerous Black Box.

**Convention:** every script used for a project task lives here, named specifically for
what it does. Each script has its own documentation file with the **same name and a
`.md` extension** (e.g. `generate-guides-index.sh` → `generate-guides-index.md`).
Don't leave task scripts in `/tmp` or run them inline-only — save them here so they can
be re-run later.

> **Completed one-off scripts live in [`archive/`](archive/README.md).** Design-system
> migration, title-block/breadcrumb standardization, one-shot content fixes, data-seeders,
> and the project rename did their job once and are kept there for reference — not part of
> any ongoing workflow. This file lists only the scripts you run as routine maintenance.
